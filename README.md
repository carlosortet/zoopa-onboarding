## Deploy

### GitHub Pages

1. Crear repositorio en GitHub
2. Push del código
3. Settings > Pages > Source: main branch
4. La web estará disponible en `https://username.github.io/zoopa-onboarding/`

### Netlify / Vercel

1. Conectar repositorio
2. Deploy automático desde main branch
3. No requiere configuración adicional
